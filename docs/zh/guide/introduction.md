# 介绍

## 特性
- 🚀 在需要之前都无需任何配置
- 📦 开箱即用
- 💪 基于 Webpack 的 Vue 应用
- 🔧 开发模式热重载
- 🔗 内置支持 `CSS` `Stylus`
- 🔨 无需配置支持 `Sass` `Less`
- 🧰 易于编写插件
